Compile kernel & qemu-linux

This is about set qemu VM with linux kernel and filesystem.

install

  • install qemu version 4.2 from source (qemu 2.0 memory allocation error?)
  • linux source download from https://mirrors.edge.kernel.org/pub/linux/kernel/v4.x/
  • install buildroot (busybox with third party compiled pkg)
    https://buildroot.org/downloads/manual/manual.html#getting-buildroot (version 2021.02.7)
