Build Hexo with github.io

It is about how to build website on my website. Bingo.

Pre-requirement

  • install git, nodejs ((https://github.com/nodejs/node-gyp#on-windows).

  • npm config (avoid using C:)

  • notice: require admin account to “npm install” with config on Windows - check!
  • windows export path: 1) systempath - NODE_PATH = F:\path\nodejs\node_modules 2) userpath - npm change to F:\path\nodejs\node_global

Install

refer to [2, 3] - check!

  • issue one: Github host key verification failed.
    • solution: input “yes” other than [enter].

migrate to a new pc

  1. git ssh config
  2. install nodejs : node -v
  3. npm install hexo-cli -g: hexo -v(path config)
  4. copy from initial pc
  5. npm install
  6. without hexo init
  7. hexo g - hexo s
  8. hexo d

more about configuration

https://ppoffice.github.io/hexo-theme-icarus/

touble shooting

  1. hexo run with root under git
  2. website img is located in /img/img under /public
